# Security Policy

## Supported Versions

Only the latest release on npm receives security fixes.

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report them privately via [GitHub Security Advisories](https://github.com/baxyz/mozlz4/security/advisories/new) or by emailing the maintainer directly.

You will receive a response within 72 hours. If the vulnerability is confirmed, a patch will be released as soon as possible.
